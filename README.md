**AI** **com**  **esp32** **wifi:**




**Objetivo** **à** **longo** **prazo**:


Utilizar um computador externo para lidar com ativadades de aprendizado de maquina.

O ESP32 enviara para o computador os dados para aprendizado.

O computador recebe e processa enviando a rede de volta ao ESP32, que só utiliza a rede (menos processamento).

------------------------------------------------------------------


**Objetivo** **à** **curto/medio** **prazo**:


Aprender as funcionalidades básicas do esp32.

Aprender a utilizar seu dualcore separadamente.

Aprender a lidar com diferentes sensores.

Aprender a lidar com interrupções.

Cameras e mapeamentos.

-------------------------------------------------------------------



**Organizaçao:**


Na primeira pasta existe um arquivo que se chama "Diario de desenvolvimento".

Todos os passos de aprendizagem, fonte de pesquisa e troubleshots estão neste arquivo.
